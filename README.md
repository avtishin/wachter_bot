# Wachter — Telegram Moderation Bot

Бот для модерации групповых чатов: просит новых участников представиться с тегом `#whois`, автоматически кикает тех, кто не представился в отведённое время.

## Быстрый старт

### Docker

```bash
docker build -t wachter_bot .
docker run -e TELEGRAM_TOKEN=<token> -e DATABASE_URL=<postgresql://...> wachter_bot
```

### Локально

```bash
pipenv install
DATABASE_URL=postgresql://localhost:5432/wachter TELEGRAM_TOKEN=<token> python wachter/bot.py
```

### Railway / облако

Установи переменные окружения:
- `TELEGRAM_TOKEN` — токен от @BotFather
- `DATABASE_URL` — строка подключения к PostgreSQL (Railway подставляет автоматически при добавлении БД)

## Как добавить бота в чат

1. Добавить бота в группу и выдать права администратора (нужны: удаление сообщений, кик участников)
2. Написать в чат сообщение с `#whois` длиной ≥ 20 символов — бот запомнит вас как участника чата
3. Написать боту в личку `/start` — откроется меню настройки чата

## Команды в чате

| Команда | Доступ | Описание |
|---|---|---|
| `/skip` | только admin | Ответ на сообщение — отменить таймер кика. Пользователь **не** записывается в БД |
| `/approve` | только admin | Ответ на сообщение пользователя **или** на приветствие бота — одобрить навсегда. Записывается в БД |
| `/whois @username` | все | Показать сохранённое представление пользователя (также `/whois <user_id>` или ответ на сообщение) |
| `/help` | все | Справка |

## Команды в личке

| Команда | Описание |
|---|---|
| `/start` | Открыть панель настройки чата |

## Как работает бот

- Новый участник получает приветствие с просьбой написать `#whois` (≥ настраиваемой минимальной длины)
- Если написал сообщение без тега — бот напомнит написать `#whois` (каждый раз)
- Если настроен таймаут — запускается таймер кика; за `notify_delta` минут до кика приходит напоминание
- Написал `#whois` (или отредактировал сообщение с тегом) → записывается в БД, таймер отменяется
- Таймер истёк → бан на `ban_duration` минут (0 = навсегда)
- При выходе из чата отправляется настраиваемое сообщение
- При повторном входе знакомый участник получает другое приветствие, таймер не запускается
