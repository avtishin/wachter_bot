# Admin Reference

## Начало работы

1. Добавить бота в чат
2. Написать в чате сообщение с `#whois` длиной ≥20 символов (бот запомнит вас как участника этого чата)
3. Написать боту в личку `/start`
4. Выбрать чат → настроить через меню

## Команды

| Команда | Где | Описание |
|---|---|---|
| `/start` | личка | Открыть панель настройки чата |
| `/skip` | чат (ответ на сообщение) | Отменить таймер кика для пользователя. Пользователь **не** записывается в БД — при следующем входе снова попросят представиться |
| `/approve` | чат (ответ на сообщение) | Одобрить пользователя навсегда. Записывается в БД — при следующем входе будет приветствоваться как знакомый, представляться не потребуется |
| `/whois <user_id>` | чат | Показать сохранённое представление пользователя |
| `/help` | любой | Справка по боту |

## Настройки чата

Все настройки меняются через `/start` → выбрать чат → нажать кнопку.

| Настройка | Описание | Значение по умолчанию |
|---|---|---|
| Таймаут кика | Минут до кика (0 = не кикать, только записывать #whois) | `0` |
| Сообщение при входе | Приветствие новому участнику | `Пожалуйста, представьтесь... У вас есть %TIMEOUT% минут.` |
| Сообщение при перезаходе | Для уже представлявшихся | `Добро пожаловать. Снова` |
| Сообщение после представления | Ответ на успешный #whois | `Добро пожаловать.` |
| Сообщение напоминания | За 10 минут до кика | `%USER_MENTION%, пожалуйста, представьтесь...` |
| Сообщение после кика | Публичное сообщение о кике (`false` или `0` — не отправлять) | `%USER_MENTION% молчит и покидает чат` |
| Regex фильтр | Регулярное выражение для фильтрации сообщений (`%TURN_OFF%` — отключить) | выключен |
| Только для новых | Применять regex-фильтр только к непредставившимся (`true`/`1` — включить) | `false` |

## Плейсхолдеры в шаблонах сообщений

| Плейсхолдер | Что подставляется |
|---|---|
| `%USER_MENTION%` | Упоминание пользователя (`@username` или ссылка на профиль) |
| `%TIMEOUT%` | Таймаут кика в минутах (`"15 мин."`) или `"не установлен"` если таймаут = 0 |

Пример:
```
Привет, %USER_MENTION%! У тебя есть %TIMEOUT%, чтобы написать #whois.
```

## Требования к #whois

- Минимальная длина сообщения: **20 символов** (включая сам тег)
- Тег `#whois` должен быть распознан как hashtag entity Telegram
- Работает только в групповых чатах (не в личке)
- Если пользователь пишет повторно — whois обновляется в БД
